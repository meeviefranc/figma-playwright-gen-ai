# Button Test Case

## Test Title
Button Visibility and Text Content

## Description
This test case checks the visibility of the 'Button' element and its text content.

## Steps
1. Navigate to the webpage containing the 'Button' element.
2. Verify that the 'left' and 'right' elements are visible on the page.
3. Verify that the 'Button' element is also visible on the page.
4. Check the text content of the 'Button' element.

## Expected Results
- The 'left', 'Button', and 'right' elements should be visible on the webpage.
- The text content of the 'Button' element should match the expected value.
